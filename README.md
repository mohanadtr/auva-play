# Auva Play
A minimal, keyboard-first video player for studying lectures.
Part of the [Auva](https://auva.dev) ecosystem.

**Live:** https://play.auva.dev

## Features
- Drag & drop local video files or paste a direct video URL
- Speed control (0.25× – 3×) with pitch correction
- A-B repeat for looping sections
- Timestamp bookmarks with notes
- Resume position per video
- Full keyboard control
- PWA — installable, works offline

## Keyboard Shortcuts
| Key | Action |
|-----|--------|
| Space | Play / Pause |
| ← / → | ±5 seconds |
| Shift + ← / → | ±30 seconds |
| [ / ] | Speed down / up |
| A / S | Set A / B point |
| F | Fullscreen |
| T | Picture-in-picture |
| ? | Show shortcuts |

## Stack
React · Vite · Tailwind CSS · PWA
